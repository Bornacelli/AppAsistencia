Cargando

Conecta Firebase

Necesitas un proyecto de Firebase para
guardar los datos en la nube.

Pasos

  1. Entra a console.firebase.google.com
  2. Crea un proyecto nuevo
  3. Activa Firestore Database (modo prueba)
  4. En Configuración → agrega una app Web
  5. Copia tus credenciales
  6. Abre index.html en un editor de texto
  7. Busca FIREBASE CONFIG y pega tus datos

Reglas de Firestore

Ve a Firestore → Reglas y pega esto:

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{doc=**} {
      allow read, write: if true;
    }
  }
}
Solo para MVP. La app final tendrá seguridad por roles.
